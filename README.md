# Rafael Oliveira's Curriculum Vitae

Static website for my Curriculum Vitae, hosted [here](https://cv.rso.pt).

## Usage

To build for production, use:

```bash
$ hugo --gc --minify
```

To launch a local development server, use:

```bash
$ hugo server
```
